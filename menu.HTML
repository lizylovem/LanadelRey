HTML

HTML (Lenguaje de Marcas de Hipertexto, del inglés HyperText Markup Language) es el código que se utiliza para estructurar y desplegar una página web y sus contenidos. Por ejemplo, sus contenidos podrían ser párrafos, una lista con viñetas, o imágenes y tablas de datos.
